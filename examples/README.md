# Examples

End-to-end example scripts and notebooks.

## Planned

- `phase0_smoke.ipynb` — minimal Colab/Jupyter that runs the Phase 0 smoke locally (no RunPod) for sanity-checking before pushing to a pod.
- `feature_browser.ipynb` — load a Qwen-Scope SAE checkpoint + an AV checkpoint; browse top-firing features with their AV descriptions side by side.
- `delta_inspector.ipynb` — Phase 3 helper: load baseline and tuned checkpoints, run the 5 feature-diff metrics, render top-drifted features with description deltas.
