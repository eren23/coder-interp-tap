# Findings

Documented research findings tied to this tap. Populated as experiments complete.

## Convention

Each finding is a Markdown file:

```
findings/
└── <YYYY-MM-DD>_<slug>.md
```

Structure:
- Title and date
- Hypothesis
- Setup (which project YAML, which W&B run, which checkpoint)
- Key result (one-paragraph headline + the numbers)
- Surprises / gotchas
- Next steps

See [`crucible-community-tap findings`](https://github.com/eren23/crucible-community-tap/tree/main/findings) for examples.
