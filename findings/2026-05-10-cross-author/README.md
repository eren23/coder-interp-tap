# Cross-Author Delta Study — 2026-05-10

End-of-day snapshot of the personalization-delta methodology after one full day of iteration. Same Qwen2.5-Coder-1.5B base, same OLD pilot SAE (24K features, layer 6), same three GitHub authors (karpathy, ggerganov, eren23), three pipeline versions ablating one knob at a time.

## What's in this folder

- `methodology_progression.png` — bar chart of `median |log2_ratio|` for each author × each version. Real W&B data.
- `lora_loss_progression.png` — same shape, LoRA SFT loss. Shows the dramatic ggerganov drop (0.63 → 0.37) when license-headered files are skipped.
- `log2_distribution_v3.png` — three histograms, one per author, showing the distribution of `log2_ratio` values across the top-500 changed features in v3.
- `top_features_v1_vs_v3_karpathy.png` — concrete feature-level comparison. v1 top-10 was dominated by `:type` annotations and `Copyright (c)` patterns; v3 top-10 is imports, assertions, and test-code structure.

All PNGs were generated by `make_plots.py` (committed in this folder) which pulls live W&B run data via the W&B Python API. To regenerate after a new round of runs, just re-run that script.
